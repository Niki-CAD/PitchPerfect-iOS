# PitchPerfect-iOS

- Description: It is a fun iOS application which transforms the voice recorded by the user to various characters like chipmunk, Darth Vader etc by changing the recorder voice's pitch and rate.

- Frameworks used:
  - UIKit
  - AVFoundation
